# Methodik

## 1. Kohortendefinition
Definition einer historischen Kohorte basierend auf:
- First Occurrence des Events „Welcome“
- Zeitraum Juni 2023
- Filter nach Land und Plattform

## 2. Funnel-Analyse
Sequenzieller Funnel mit fünf Schritten.
Breakdown nach Plattform.

## 3. Time-to-Convert
Berechnung der durchschnittlichen Zeit vom Welcome bis zum ersten Play-Event.

## 4. Retention
Unbounded Retention von Welcome zu:
- Any Active Event
- Play Song or Video

## 5. Journey Mapping
Visualisierung der häufigsten Nutzerpfade bis zum ersten Content-Konsum.
