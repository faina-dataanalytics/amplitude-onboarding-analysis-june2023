# Projektkontext

Dieses Projekt basiert auf dem Amplitude Demo-Datensatz „Media Streaming – Analytics“.

Das Szenario simuliert ein reales Streaming-Unternehmen, das das Onboarding neuer mobiler Nutzer optimieren möchte.

Die Analyse wurde mit Fokus auf:
- Produktaktivierung
- Early Retention
- Time-to-Value
durchgeführt.
